##Instalation\
\
#Step 0\
1. Install nvm here https://github.com/coreybutler/nvm-windows \
2. Create a Supabase and Vercel account\
3. install git for version control\
\
#Step 1: Clone or fork the repository\
The fork can be done directly on GitHub, or you can use this command to clone it:\
git clone https://github.com/OrdinaryBagel/inst377-final-medicine-scheduler.git \
\
#Step 2: Install packages\
The commands below will install all required dependencies\
cd inst377-final-medicine-scheduler\
npm install\
\
#Step 3: API keys\
You will need 3 2 from Supabase and 1 from the NYT API\
These should be in a .env file\
VERY IMPORTANT: the .env should be in .gitignore so you do not accidentally publish your api keys\
