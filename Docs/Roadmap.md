##Roadmap
There are 4 current goals for the future of this project in descending importance
#1. Security Improvements
This app is based in healthcare and health informatics making user privacy extremely important which makes this app's complete lack of it very detrimental
#2. Visual Improvments
The visuals of the site are very lackluster and stand to gain the most improvement
#3. Improving Calendar Interactability
The only way to interact with the calendar directly is to mark when you forgot to take medicine which while useful, leaves a lot of room for improvements
