## About the App

Medicine Scheduler is a web application deployed on Vercel that uses Supabase as its database and allows users to:

-   Schedule when they need to take their medicine (daily, weekly, or monthly cycles)
-   View their medication schedule on a dynamically updating calendar
-   Receive alerts about drug recalls and shortages of their medications via the OpenFDA API
-   Read health-related news from the New York Times

URL: [inst377-final-medicine-scheduler.vercel.app](https://inst377-final-medicine-scheduler.vercel.app)
